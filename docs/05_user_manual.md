## 1. 快速入门

### 1.1 访问系统

打开浏览器, 访问系统地址:
- 本地开发: <http://127.0.0.1:8000>
- 生产环境: <http://your-domain.com>

### 1.2 登录

1. 在登录页输入用户名与密码
2. 点击"登录"按钮
3. 登录成功后自动跳转到数据看板

![登录页](https://via.placeholder.com/600x300?text=登录界面)

### 1.3 注册新账号(若开放)

1. 在登录页点击"立即注册"
2. 填写用户名、邮箱、密码, 选择角色(查看者或库管员)
3. 点击"注册"按钮, 系统提示"注册成功"

> 注意: 注册时只能选择"查看者"或"库管员", "系统管理员"需要联系现有管理员开通。

### 1.4 角色权限对照

| 操作 | 系统管理员 | 库管员 | 查看者 |
|------|:---:|:---:|:---:|
| 查看所有数据 | ✅ | ✅ | ✅ |
| 新增/编辑/删除商品 | ✅ | ✅ | ❌ |
| 出入库操作 | ✅ | ✅ | ❌ |
| 库存盘点 | ✅ | ✅ | ❌ |
| 用户管理与角色分配 | ✅ | ❌ | ❌ |
| 登录 Django 后台 | ✅(超管) | ❌ | ❌ |

## 2. 数据看板

数据看板是系统首页, 提供整体运营概览。

### 2.1 KPI 指标卡片(顶部)

| 指标 | 说明 |
|------|------|
| 在用商品总数 | 所有"在用"状态的商品 SKU 数 |
| 库存总价值 | 全部商品按成本价计算的库存总金额 |
| 今日入库 | 当天所有入库流水的数量合计 |
| 今日出库 | 当天所有出库流水的数量合计 |

### 2.2 出入库趋势图

- 默认展示近 30 天的入库 / 出库趋势
- 点击右上角下拉框可切换"近 7 天 / 近 30 天 / 近 90 天"
- 绿色为入库, 红色为出库

### 2.3 分类分布饼图

按商品分类展示数量分布, 鼠标悬停可查看具体数值。

### 2.4 低库存预警

- 列出当前库存 ≤ 警戒线的前 10 个商品
- 表格背景为黄色高亮, 点击商品名可跳转详情页进行补货
- 若提示"一切正常 ✅", 说明没有低库存商品

### 2.5 最近操作

显示最近 10 条系统操作日志, 包括登录、出入库、商品增改等。

## 3. 商品管理

### 3.1 查看商品列表

点击顶部菜单"商品管理", 进入商品列表。

#### 搜索与过滤

- **关键字**: 输入 SKU 或商品名称模糊搜索
- **分类**: 下拉选择特定分类
- **仅低库存**: 勾选后只显示库存不足的商品
- 点击"搜索"按钮应用筛选

### 3.2 新增商品(库管员/管理员)

1. 点击右上角"新增商品"
2. 填写商品信息:
   - **SKU 编号**(必填): 唯一, 建议格式如 `P0001`
   - **商品名称**(必填)
   - **所属分类**(必填)
   - **规格型号**: 如 "1.5L 红色"
   - **计量单位**: 件/箱/千克等
   - **销售单价**(必填)
   - **成本单价**: 用于计算库存价值
   - **安全库存**: 低于此值系统会预警
   - **商品图片**: 上传 JPG/PNG
   - **备注**: 任意补充信息
3. 点击"保存"

### 3.3 编辑商品

在商品列表点击黄色"铅笔"图标, 修改字段后保存。

### 3.4 查看商品详情

点击商品名称进入详情页, 可查看:
- 商品基本信息与图片
- 当前库存与警戒状态
- 库存价值(成本 × 数量)
- 最近 20 条出入库流水

### 3.5 删除商品

1. 点击红色"垃圾桶"图标进入确认页
2. 阅读提示, 点击"确认删除"

> 注意: 若商品有关联的出入库流水, 系统将"保护性删除"商品(实际删除但流水保留)。

### 3.6 分类管理

在"分类管理"页面可以新增、编辑、删除分类。

> 删除限制: 若分类下仍有商品, 系统会拒绝删除, 需先转移或删除商品。

## 4. 库存出入库

### 4.1 查看流水

点击"出入库"菜单, 默认显示最近所有流水, 包含:
- 时间、类型(入库/出库)
- 商品、数量、单价、总金额
- 操作人、备注

#### 筛选

- 关键字: SKU/名称/备注
- 类型: 入库 / 出库
- 开始日期 / 结束日期

页面顶部会显示当前筛选范围内的入库 / 出库合计。

### 4.2 新增入库/出库

1. 点击"新增出入库"
2. 选择商品(下拉显示所有"在用"商品)
3. 选择类型:
   - **入库**: 增加库存(如采购到货、退货入库)
   - **出库**: 减少库存(如销售出库、领用)
4. 填写数量(必须 ≥ 1)
5. 填写单价(入库填成本价, 出库填销售价)
6. 填写备注(可选)
7. 点击"提交"

**业务校验**:
- 出库时数量不能超过当前库存, 系统会提示具体差额
- 提交成功后, 商品库存立即更新

### 4.3 撤销流水(误操作恢复)

1. 在流水列表点击红色"撤销"按钮
2. 确认撤销后, 系统会:
   - 反向调整库存(入库流水 → 减库存; 出库流水 → 加库存)
   - 删除该条流水记录
3. 同时记录操作日志

> 建议: 撤销仅用于纠正误操作, 正常业务波动应通过新增反向流水处理。

### 4.4 查看流水详情

点击蓝色"眼睛"图标, 可查看流水的完整信息。

## 5. 库存盘点

### 5.1 何时需要盘点

- 周期性月度 / 季度 / 年度盘点
- 怀疑库存账实不符
- 商品损耗或丢失核查

### 5.2 新增盘点记录

1. 点击"盘点"菜单 → "新增盘点"
2. 选择商品
3. 填写**实际盘点库存**(实际数到的数量)
4. 填写备注(可选, 如"2026 Q2 季度盘点")
5. 点击"提交盘点"

**自动处理逻辑**:
- 系统自动记录当前"系统库存"作为对比基准
- 自动计算差异 = 实际库存 - 系统库存
- 若差异 ≠ 0:
  - 自动生成一条出入库流水(类型由差异正负决定)
  - 自动同步商品库存为实际值

### 5.3 查看盘点历史

盘点列表展示所有盘点记录, 差异列:
- 绿色 `+N`: 实际比系统多 N(系统少记了)
- 红色 `-N`: 实际比系统少 N(可能有损耗或丢失)
- 灰色 `0`: 账实相符

## 6. 用户与权限管理

> 仅系统管理员可见此模块。

### 6.1 查看用户列表

点击"用户管理", 可查看所有用户的基本信息、角色、账号状态。

### 6.2 修改用户角色

1. 点击"编辑"按钮
2. 修改角色(系统管理员 / 库管员 / 查看者)
3. 可同时修改邮箱、姓名
4. 保存

> 注意: 不能修改超级管理员(is_superuser=True)的角色。

### 6.3 启用 / 停用账号

- 点击"停用"按钮: 该用户无法再登录
- 点击"启用"按钮: 恢复登录权限

> 不直接删除用户, 是为了保留其历史操作记录的关联。

## 8. 常见问题

### Q1: 忘记密码怎么办?
A: 联系系统管理员, 通过"用户管理"页面无法直接重置, 管理员可登录 Django 后台 `/admin/` 重置。

### Q2: 为什么我看不到"新增商品"按钮?
A: 您当前角色是"查看者", 仅具备只读权限。如需编辑权限, 请联系管理员调整角色。

### Q3: 出库时报"库存不足", 但我看到明明有库存?
A: 可能是其他人正在并发操作同一商品。请刷新页面查看最新库存, 或联系管理员核对流水。

### Q4: 删除商品后流水还在, 数据库会不会越来越大?
A: 是的, 出于审计目的流水永久保留。建议每年归档 1 年以上的历史数据。

### Q5: 上传的商品图片在哪里?
A: 默认保存在项目 `media/products/` 目录, 通过 Nginx 直接访问。生产环境建议定期备份此目录。

### Q6: 如何导出商品列表到 Excel?
A: 当前版本未提供, 可通过 Django Admin 进入 `/admin/products/product/`, 使用浏览器打印或第三方插件导出。后续版本会提供 CSV 导出功能。

### Q7: 可以多个人同时操作系统吗?
A: 可以, 系统使用数据库事务与行级锁保证并发安全。出库场景会通过 `SELECT FOR UPDATE` 避免超扣。
